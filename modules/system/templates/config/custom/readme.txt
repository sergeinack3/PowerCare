La spec de configuration par etablissement/service custom permet de préciser un tpl à afficher.
Elle permet de faire une config personnalisée, avec l'affichage qu'on veut.
(par exemple les configs des constantes, qui contiennent en fait 3 configs : rang par formulaire, par graphe, et la couleur dans les graphiques)

Dans les specs de la config, il faut mettre 'custom' comme type, et préciser dans l'option 'tpl' le nom du template, qui doit impérativement être dans le répertoire system/templates/config/custom.

Il est possible de préciser ses propres options, selon les besoins.
